# V0435 - Welke taakstatussen gebruiken we?

## Vraagstuk

Oorspronkelijk idee:
- `Te doen`
- `Gedaan`

Maar het kan zijn dat taken vervallen.

Alternatief 1:
- `Te doen`
- `Afgedaan`
Waarbij afgedaan zowel 'het is gedaan' als 'het hoeft niet meer' kan betekenen.

Alternatief 2:
- `Te doen`
- `Gedaan`
- `Vervallen`
